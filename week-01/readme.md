# webDev - HTML Fundamentals Project

## Objective
The objective of this project was to reinforce basic HTML skills by building a structured, multi-page website from scratch. The focus was on mastering document organization, creating functional navigation between pages, and implementing multimedia elements like images and lists.

## What Was Done
For this project, a 4-page website was developed with the following features:
* **Proper Document Structure:** Each page (`index.html`, `about.html`, `contact.html`, and `external.html`) includes the `<!DOCTYPE html>`, `<html>`, `<head>`, and `<body>` tags.
* **Meta Tags & Titles:** Metadata was added for character encoding and responsive viewport settings.
* **Navigation System:** A consistent navigation menu was added to the top of every page to allow seamless movement between internal pages.
* **Multimedia Integration:** * An online image was pulled from Unsplash for the Home page.
    * A local image placeholder was added to the Resources page.
* **Content Organization:** * Used an **unordered list** on the About page to show learning goals.
    * Used an **ordered list** on the Contact page to list communication methods.
* **External Linking:** Added a functional link to the MDN Web Docs to practice linking to outside resources.

## Challenges Faced
* **Relative Pathing:** One challenge was ensuring that the internal links worked correctly across all four files, especially making sure the "Home" link always pointed back to `index.html`.
* **Image Implementation:** Learning the difference between linking an image via a URL versus a local file path required careful attention to the `src` attribute.
* **Code Consistency:** Keeping the navigation bar identical on every page so the user experience remains smooth.
